currentPath ( $  filename  ) 

Returns filename relative to currently executing script.


Example: include currentPath('images/a.jpg');
Example return: include '../images/a.jpg';
Just counts number of slashes (-1) in PHP_SELF and prepends that many '../' to the $filename. Very useful when including a script from an already included one (that could be included from different directories).

Parameters:
<type> $filename Filename relative to the document root.
