Detailed Description

A manager for profile.php, handles registration and profile changes.

Messages to the user and values they've filled are carried in session and then put to $profileMessage, $lastMail, $lastName, $lastAddress. To make sure these strings are displayed only once, they are unset on every page load, just after they are shown to the user. This is for user's comfort, not to have to fill particular form fields again. Entered password is not pre-filled, naturally.
