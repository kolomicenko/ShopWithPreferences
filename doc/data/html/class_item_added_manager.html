Detailed Description

The job of this class is adding items to the shopping cart.

It adds an item to the cart when POST is set. Otherwise, when GET[id] is set, display items linked to the item with that id. That means when user adds an item to the cart, this manager is first created with POST[id] and then it redirects to itself with GET[id], so that it can show the linked (or related) items.
