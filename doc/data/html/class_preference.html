Detailed Description

Base class for all preferences.

Preferences have to inherit from this class and implement setPreference() and getPreference(). BlackBox class allways handles preferences and calls set and get functions. These functions' parameters are arrays, because every preference accepts different number of arguments. Constructors of preference classes can take just one parameter, which is userID of current user.
