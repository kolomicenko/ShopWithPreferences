Detailed Description

Parent of all managers that use categories.

Works with UserCategories class and displays them in left menu. Category tree is browsed using php, not javascript, so that every click inside left menu means loading another page. This has the advantage of listing products from a category and all its descendants at a time, not just products from a leaf category, when user gets to it.

Constructor expects ID of the current category in GET or at least as the parameter of itself. In other cases, the ID is set to zero and the home page is displayed.
